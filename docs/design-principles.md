# 页面设计原则（去 AI 感）

## 1) 文案必须具体

- 优先写真实项目、真实动作、真实结果。
- 避免空泛表达，例如“热爱技术”“追求卓越”。

## 2) 视觉层级克制

- 通过字号、间距、分组建立层级，不依赖复杂特效。
- 优先可读性，不做大面积高饱和渐变背景。

## 3) 组件数量有节制

- 首页每屏聚焦一个核心目标，避免卡片堆叠。
- 每个区块只保留一个主要信息点。

## 4) 交互动效不过度

- 动效只用于状态反馈，不用于炫技。
- 保持动效短且轻量，不影响阅读节奏。

## 5) 验收标准

- 随机替换文案后，页面仍自然、可信、可读。
- 不出现“模板拼装”或“生成式口号”痕迹。
